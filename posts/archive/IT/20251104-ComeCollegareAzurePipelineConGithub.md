I fondamentali non passano mai di moda. In un recente ingaggio di consulenza abbiamo migrato repository TFVC e git da un Team Foundation Server on-prem a GitHub e gestione del lavoro (work-item) su Azure DevOps. In questo video video mostro le funzionalità di "dialogo" tra Azure DevOps e GitHub con approfondimento sulle Azure Pipeline e come possono essere usate per proteggere i branch su GitHub.

## Punti salienti

Nel video viene trattato in particolare:

- Come connettere le Azure Pipeline a GitHub
- Le funzionalità che si abilitano (status check)
- Protezione dei branch tramite status check delle pipeline per impedire modifiche non approvate ai branch di lunga vita
